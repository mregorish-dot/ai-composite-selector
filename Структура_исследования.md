# Структура исследовательской работы

## Тема
**"Применение ИИ и цифровых технологий для выбора композита и проведения реставраций жевательных зубов прямым методом с учётом данных ЭМГ и технических характеристик композита у пациентов с аномалиями прикуса"**

---

## Цель исследования

Разработка и валидация системы на основе искусственного интеллекта для автоматизированного выбора оптимального композитного материала для реставрации жевательных зубов прямым методом на основе:
1. Электромиографических данных (ЭМГ) жевательных и височных мышц
2. Технических характеристик композитных материалов
3. Клинических особенностей пациентов с аномалиями прикуса

---

## Задачи исследования

### 1. Аналитическая часть
- [ ] Систематизация данных о технических характеристиках композитных материалов
- [ ] Анализ корреляций между ЭМГ-показателями и выбором композита
- [ ] Разработка критериев классификации композитов по пригодности для жевательных зубов
- [ ] Анализ различий ЭМГ-показателей между различными аппаратами

### 2. Разработка ИИ-модели
- [ ] Создание базы данных композитных материалов с техническими характеристиками
- [ ] Разработка алгоритма нормализации ЭМГ-данных для разных аппаратов
- [ ] Построение модели машинного обучения для выбора композита
- [ ] Валидация модели на клинических данных

### 3. Клиническая часть
- [ ] Проведение ЭМГ-исследований у пациентов с аномалиями прикуса
- [ ] Оценка эффективности выбранных композитов
- [ ] Сравнение результатов ИИ-рекомендаций с клиническими решениями

---

## Методология

### 1. Критерии включения пациентов
- Пациенты с аномалиями прикуса
- Необходимость реставрации жевательных зубов прямым методом
- Отсутствие противопоказаний к ЭМГ-исследованию

### 2. ЭМГ-протокол
- **Аппарат:** [указать используемый аппарат - Synapsys/Колибри/другой]
- **Условия измерения:**
  - При акте жевания (средняя амплитуда)
  - При максимальном сжатии челюстей (максимальная амплитуда)
- **Мышцы:** жевательная и височная (правая и левая)
- **Нормализация:** относительно контрольных значений или индивидуальных базовых показателей

### 3. Классификация композитов

**Критерии исключения:**
- Low Viscosity композиты (полимеризационная усадка до 6%)
- Композиты с усадкой >3% для жевательных зубов
- Композиты с микротвердостью <50 KHN

**Критерии включения:**
- High Viscosity Bulk Fill композиты
- Полимеризационная усадка: 1-3%
- Микротвердость: ≥50 KHN (предпочтительно >70 KHN)
- Концентрация наполнителя: 25-50%
- Высокая износостойкость

### 4. ИИ-модель

**Входные данные:**
- ЭМГ-показатели (нормализованные)
- Тип аппарата ЭМГ
- Клинические данные пациента (возраст, тип аномалии прикуса, степень стираемости)
- Технические характеристики доступных композитов

**Выходные данные:**
- Рекомендуемый композит
- Вероятность успешности реставрации
- Обоснование выбора

**Алгоритм:**
- Предобработка и нормализация ЭМГ-данных
- Извлечение признаков (features)
- Обучение модели (Random Forest / Gradient Boosting / Neural Network)
- Валидация и тестирование
