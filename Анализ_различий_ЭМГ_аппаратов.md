## Ключевые различия и выводы

### 1. Различия в абсолютных значениях

**При акте жевания (средняя амплитуда):**
- **Synapsys**: жевательная ~339–351 мкВ, височная ~235–243 мкВ
- **Колибри**: жевательная ~111 мкВ, височная ~138 мкВ

**Разница:**
- Жевательная мышца: **Synapsys показывает значения в ~3 раза выше** (339–351 мкВ vs 111 мкВ)
- Височная мышца: **Synapsys показывает значения в ~1,7 раза выше** (235–243 мкВ vs 138 мкВ)

### 2. Различия при максимальном сжатии

**Synapsys (максимальное сжатие):**
- Жевательная: 351–360 мкВ
- Височная: 248–275 мкВ

**Колибри (максимальная амплитуда):**
- Жевательная: 278 ± 118 мкВ
- Височная: 558 ± 126 мкВ

**Разница:**
- Жевательная мышца: значения сопоставимы (Synapsys: 351–360 мкВ, Колибри: 278 мкВ)
- Височная мышца: **Колибри показывает значения в ~2 раза выше** (558 мкВ vs 248–275 мкВ)

### 3. Возможные причины различий

1. **Разные методики измерения:**
   - Различия в настройках усиления сигнала
   - Различия в частоте дискретизации
   - Различия в обработке сигнала (фильтрация, усреднение)

2. **Разные протоколы исследования:**
   - Различные условия записи
   - Различные позиции электродов
   - Различные инструкции пациентам

3. **Разные популяции:**
   - Различные возрастные группы
   - Различные клинические состояния (норма vs патология)

### 4. Рекомендации для исследования

1. **Использовать один аппарат** для всех измерений в рамках одного исследования
2. **Калибровка между аппаратами** при необходимости сравнения данных
3. **Нормирование показателей** относительно индивидуальных базовых значений пациента
4. **Учет относительных изменений** (процентные изменения) вместо абсолютных значений при сравнении между исследованиями

---

## Выводы для практического применения

1. **Абсолютные значения ЭМГ не могут быть напрямую сопоставлены** между разными аппаратами без калибровки
2. **Относительные изменения** (процентные приросты/снижения) более надежны для сравнения
3. **Важно стандартизировать протокол** измерения для воспроизводимости результатов
4. **Для ИИ-модели** необходимо учитывать тип аппарата как признак или нормализовать данные относительно контрольных значений
