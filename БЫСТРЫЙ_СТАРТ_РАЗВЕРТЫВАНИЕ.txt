1. Откройте: https://github.com/new

2. Заполните:
   • Repository name: composeai-app
   • Description: ComposeAI - ИИ-система выбора композита
   • Visibility: Public (важно!)
   • НЕ ставьте галочки на README, .gitignore, license

3. Нажмите "Create repository"

ВАРИАНТ B (Вручную):
   Выполните в терминале:

   cd "/Users/egorryzhkov/Desktop/AI Double"
   git init
   git add .
   git commit -m "Initial commit: ComposeAI"
   git branch -M main
   git remote add origin https://github.com/ВАШ_USERNAME/composeai-app.git
   git push -u origin main

   ⚠️ Замените ВАШ_USERNAME на ваш GitHub username!

🚀 ШАГ 3: РАЗВЕРНИТЕ НА STREAMLIT CLOUD
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Откройте: https://share.streamlit.io/

2. Войдите через GitHub (кнопка "Sign in with GitHub")

3. Нажмите "New app"

4. Заполните:
   • Repository: ВАШ_USERNAME/composeai-app
   • Branch: main
   • Main file path: app/app.py

5. Нажмите "Deploy"

6. Подождите 1-2 минуты...
