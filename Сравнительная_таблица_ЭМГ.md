## Рекомендации для ИИ-модели

### Стратегия 1: Нормализация относительно контрольных значений

```
Нормализованное значение = (Измеренное значение / Контрольное значение) × 100%
```

**Преимущества:**
- Устраняет различия между аппаратами
- Позволяет сравнивать данные из разных источников

**Недостатки:**
- Требует наличия контрольных значений для каждого аппарата

### Стратегия 2: Использование относительных изменений

```
Относительное изменение = ((Значение после - Значение до) / Значение до) × 100%
```

**Преимущества:**
- Не зависит от абсолютных значений аппарата
- Фокус на динамике изменений

### Стратегия 3: Включение типа аппарата как признака

**Входные данные для ИИ:**
- ЭМГ-показатели (сырые значения)
- Тип аппарата (категориальный признак)
- Нормализованные значения (относительно контроля)

**Преимущества:**
- Модель может учитывать специфику каждого аппарата
- Более гибкий подход

---

## Практические рекомендации

1. **Для текущего исследования:**
   - Использовать **один аппарат** для всех измерений
   - Если необходимо сравнение с литературой - использовать **нормализованные значения**

2. **Для ИИ-модели:**
   - Включить **тип аппарата** как признак
   - Использовать **относительные изменения** вместо абсолютных значений
   - Применить **стандартизацию** (z-score) для каждого типа аппарата отдельно

3. **Для публикации:**
   - Указать тип аппарата в методе
   - Обсудить ограничения при сравнении с данными других исследований
   - Предложить методологию нормализации для будущих исследований

```
Нормализованное_значение = (Измеренное_значение / Контрольное_среднее) × 100%
```
